# Lottery

> An electron-vue project for lottery

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build for linux
npm run build:linux

# build for osx
npm run build:darwin

# ensure on windows build, or wine in linux with electron windows package in ~/.electron
npm run build:win32
```
